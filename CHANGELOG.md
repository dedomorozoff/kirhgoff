# История изменений

## [v1.1] - 2025-12-06

### Добавлено
- **Полная реализация проверки 2-го закона Кирхгофа (KVL)**:
  - Автоматический расчет суммы напряжений в выбранном контуре
  - Визуализация направления обхода контура стрелками
  - Нумерация компонентов в порядке обхода (1, 2, 3...)
  - Проверка выполнения закона (Σ U ≈ 0)
  - Индикация правильности выбранного контура (✓ или ⚠)
  - Кнопка "Очистить контур" для сброса выбора

### Улучшено
- Информационная панель теперь показывает:
  - Падение напряжения на каждом компоненте с направлением (→/←)
  - Суммарное напряжение в контуре с точностью до 4 знаков
  - Цветовую индикацию результата проверки
- Обновлена справка с инструкциями по использованию KVL
- Визуальная подсветка контура янтарным цветом (#d97706)

### Технические детали
- Добавлена функция `updateKVLOverlay()` с полным расчетом
- Улучшена отрисовка контура в функции `draw()`:
  - Пунктирные линии между компонентами
  - Стрелки направления обхода
  - Нумерация компонентов
- Обработчики событий для управления состоянием кнопок
